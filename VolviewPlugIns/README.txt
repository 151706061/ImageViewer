
ITK plug-ins for VolView
------------------------

This directory contains code for generating run-time plug-ins for the
visualization application VolView. 

Detailed information about Volview is available at:

http://www.kitware.com/products/volview.html

A lite version of VolView will be publicly available this year (2003).

This plug-ins make part of an ongoing effort for integrating ITK in
academic and commercial applications in order to extend the benefits
of the toolkit to the final users involved in medical image processing
applications.
