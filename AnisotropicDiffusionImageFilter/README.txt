This example shows how to take in a 2D scalar PNG image and filter
it using Perona-Malik style anisotropic diffusion.

Use:

itk2DAnisotropicDiffusionImageFilter input.png output.png number_of_iterations conductance_term

Note that this example does NOT handle COLOR PNG images.

See the documentation for itkAnisotropicDiffusionImageFilter for more information.
